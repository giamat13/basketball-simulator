# 🏀 כדורסל - אקשן מהיר (Fast-Action Basketball)

## ✨ תכונות עיקריות

* **🏆 מערכת טורנירים:** התמודדות מול 9 מדינות שונות ברמות קושי משתנות, מארה"ב ועד איטליה.
* **💰 חנות שדרוגים (Upgrades):** צבירת כסף מניצחונות והשקעה בשיפור המהירות, כוח הזריקה, הדיוק, הסטמינה וההגנה.
* **🎮 מכניקת משחק מתקדמת:**
    * מד כוח לזריקות מדויקות.
    * מערכת חטיפות (Steals) אקטיבית.
    * חישוב נקודות לפי מרחק (2 או 3 נקודות).
    * אינטליגנציה מלאכותית ליריב שמתאימה את עצמה לרמת הקושי.
* **📱 עיצוב רספונסיבי:** התאמה מלאה למסכים בגדלים שונים.

## 🕹️ איך משחקים?

### מקשים ובקרות:
* **תנועה:** מקשי `WASD` או **חצים**.
* **ריצה מהירה (Sprint):** מקש `SHIFT` שמאלי.
* **כיוון:** הזזת ה**עכבר** על גבי המגרש.
* **זריקה לסל:** לחיצה ממושכת על ה**לחצן השמאלי בעכבר** (לטעינת כוח) ושחרורו.
* **חטיפת כדור:** מקש **רווח (Space)** כאשר נמצאים בקרבת היריב.

### חוקי המשחק:
1. בחר יריב מתפריט הפתיחה.
2. לרשותך 4 דקות להשיג כמה שיותר נקודות.
3. ניצחון מזכה בפרס כספי בהתאם לרמת הקושי של היריב.
4. כל 3 ניצחונות תעלה רמה (Level UP).
